VBA

标识符

命名规则

  1. 字母打头，由字母，数字，下划线组成
  2. 字符长度小于40（Excel2002以上中文版本等，可以用汉字且长度可达254个字符）
  3. 不能与保留字重名：public， private，dim，goto，next， with，integer，single等

运算符

  1. 赋值运算符：=
  2. 数学运算符：&， +（字符链接符），+（加），-（减）， Mod（取余），\（整除），*（乘），/（除），-（负），^（指数）
  3. 逻辑运算符：Not，And，Or，Xor，Eqv，lmp（隐含）
  4. 关系运算符： =， <>（不等于），>，<，>=，<=， Like，ls
  5. 位运算符：Not，And，Or，Xor，Eqv，lmp

数据类型

VBA有12种数据类型，用户还可以根据一下类型用Type自定义数据类型

数据类型 类型标识符 字节
字符串型 String $ 字符长度(0-65400)
字节型 Byte 无 1
布尔型 Boolean 无 2
整数型 Integer % 2
长整数型 Long & 4
单精度型 Single ! 4
双精度型 Double # 8
日期型 Date 无 8 公元100/1/1-99/12/31
货币型 Currency @ 8
小数点型 Decimal 无 14
变体型 Variant 无 以上任意类型可变
对象型 Object 无 4

变量与常量

  1. VBA允许使用未定义的变量,默认是变体变量

  2. 再模块通用说明部分,加入Option Explicit 语句可以强迫用户进行变量定义

  3. 变量定义语句以及变量作用域
    Dim 变量 as 类型 定义为局部变量 Dim xyz as integer
    Private 变量 as 类型 定义为私有变量 Private xyz as byte
    Public 变量 as 类型 定义为公有变量 Public xyz as single
    Global 变量 as 类型 定义为全局变量 Global xyz as date
    Static 变量 as 类型 定义为静态变量 Static xyz as double

  4. 常量作为变量的一种特例,用Const定义, 且定义时赋值, 程序中不能该百年值,作用域也如同变量作用域, 如: Const Pi = 3.1415926 as single

数组

数组是包含相同数据类型的一组变量的集合, 对数组中单个变量引用通过数组索引下标进行, 再内存中表现为一个连续的内存块,必须用Global或Dim语句来定义

二维数组按行列排列: XYZ(行, 列)

VBA支持动态数组,定义时无大小维数声明,再程序中再利用Redim语句来重新改变数组大小,原来数组内容可以通过加preverse关键字来保留

注释和赋值语句

VB中有两种方法标识为注释语句

  1. 注释
    1. 单引号: '可以位于别的语句后面,也可以单独一行
    2. Rem 只能单独一行
  2. 赋值语句
    1. 赋值语句是进行变量或对象属性赋值的语句,采用复制号=
    2. 对对象的赋值采用:set myobject = object 或 myobject:=object

书写规范

  1. VBA不区分标识符的字母大小写,一律认为是小写字母
  2. 一行可以书写多条语句,各语句之间以冒号分开
  3. 一条语句可以多行书写,以空格加下划线 _来表示下行为续行
  4. 标识符最好简洁明了,不造成歧义

判断语句

  1. If ... Then ... Else语句

    If condition Then [statements][Else elsestatements]

    eg: If A>B And C>D Then A=B+2 Else A=C+2

  2. Select Case ... Case ... End Case 语句

  3. Choose函数

    choose(index, choice-1, choice-2,…,choice-n)可以用来选择自变量串列中的一个值,并将其返回,index必要参数,数值表达式或字段,它的运算结果是一个数值,且界于1和可选择的项目数间,choice必要参数,Variant表达式,包含可选择项目的其中之一,GetChoici = Choose(Ind, “Speedy”, “United”, “Federal”)

  4. Switch函数

    Switch(expr-1, value-1[, expr-2, value-2 [, expr-n, value-n]]) Switch函数和Choose函数类似,但是是以两个一组的方式返回要的值,再串列中最先为TRUE的值会被返回,expr必要参数,要加以Variant表达式,value必要参数,如果相关的表达式为True,则返回此部分的数值或表达式,没有一个表达式为True Switch会返回一个Null值

其他类语句和错误语句处理

  1. 其他循环语句

    1. Goto line 跳转到line行

    2. On expression gosubl destinationionlist 或者 on expression goto destinationlist 语句为更具expression 表达式的值来跳转到索要的行号或行标记

    3. Gosub line … line … Return， Return返回到Gosub line行

    4. while..wend语句，只要条件为TRUE，循环就执行，是VB老语法留下的，

  2. 错误语句处理

    执行阶段会有错误的情况发生,利用OnError语句来处理错误,启动一个错误的处理程序

过程与函数

过程时构成陈虚谷的一个模块,往往用来完成一个相对独立的功能,过程可以时陈虚谷更加清晰,更有结构性,VBA有四种过程:Sub过程,Function函数,Property属性过程和Event时间过程

  1. Sub过程

    Sub过程的参数两种传递方式:按值传递(ByVal)和按地址传递(ByRef)

  3. Property属性过程和Event事件过程

    VB在对象功能上添加的两个过程,与对象特征密切相关

内部函数

  1. 测试函数
    • IsNumeric(x) :是否为数字,返回Boolean结果
    • IsDate(x) :是否为日期,返回Boolean结果
    • IsEmpty(x): 是否为Empty,返回Boolean结果
    • IsArray(x): 指出变量是否为一个数组
    • IsError(Expression): 指出表达式是否为一个错误值
    • IsNull(expression): 指出表达式是否不包含任何有效数据
    • IsObject(identifier): 指出标识符是否表示对象变量
  2. 数学函数
    • Sin(x),Cos(x),Tan(x),ATan(x) 三角函数,单位为弧度
    • Log(x)返回x的自然对数
    • Exp(x)返回e的x次方
    • Abs(x)返回绝对值
    • Int(number),Fix(number)返回参数的整数部分,Int向下取证,Fix向上取整
    • Sgn(number) 返回一个Variant(Integer), 指出参数的正负号
    • Sqr(number)返回一个Double,指定参数的平方根
    • VarType(varname) 返回一个Integer,指出变量的子类型
    • Rnd(x) 返回0-1之间的单精度数据,x为随机种子
  3. 字符串函数
    • Trim(string) 去掉string左右两端空白
    • Ltrim(string) 去掉string左端空白
    • Rtrim(string) 去掉string右端空白
    • Len(string) 计算string长度
    • Left(string, x) 取string左段x个字符组成的字符串
    • Right(string, x) 取string右段x个字符组成的字符串
    • Mid(string, start, x) 取string从start位开始的x个字符组成的字符串
    • Ucase(string) 转换为大写
    • Lcase(string) 转换为小写
    • Space(x) 返回x个空白的字符串
    • Asc(string) 返回一个integer, 代表字符串中首字母的字符代码
    • Chr(charcode) 返回string, 其中包含有与指定的字符代码相关的字符
  4. 转换函数
    • CBool(expression) 转换为Boolean型
    • CByte(expression) 转换为Byte型
    • CCur(expression) 转换为Currency型
    • CDate(expression) 转换为Date型
    • CDbl(expression) 转换为Double型
    • CDec(expression) 转换为Decemal型
    • CInt(expression) 转换为Integer型
    • CLng(expression) 转换为Long型
    • CSng(expression) 转换为Single型
    • CStr(expression) 转换为String型
    • CVar(expression) 转换为Variant型
    • Val(string) 转换为数据型
    • Str(number) 转换为String
  5. 时间函数
    • Now 返回一个Variant(Date),根据计算机系统设置的日期和时间来指定日期和时间
    • Date 返回包含系统日期的Variant(Date)
    • Time 返回一个指明当前系统时间的Variant(Date)
    • Timer 返回一个Single, 代表从物业开始到现在经过的秒数
    • TimeSeial(hour, minute, second) 返回一个Variant(Date), 包含具体的时,分,秒的时间
    • DateDiff(interval, date1, date2[,firstdayofweek[,firstweekofyear]]) 返回Variant(Long)的值,表示两个指定日期间的时间间隔数目
    • Second(time) 返回一个Variant(Integer) ,值为0到59之间的整数,表示一分钟之间的某个秒
    • Minute(time) 返回一个Variant(Integer), 值为0到59之间的整数
    • Hour(time) 返回一个Variant(Integer), 值为0到23之间的整数
    • Day(date) 返回一个Variant(Integer), 值为1到31之间的整数
    • Month(date) 返回一个Variant(Integer), 值为1到12间的整数
    • Year(date) 返回Variant(Integer), 包含表示年份的整数
    • Weekday(date, [firstdayofweek]) 返回一个Variant(Integer) 包含一个整数,代表某个日期时星期几

文件操作

  • 文件Dir[(pathname[,attributes])]
    • pathname 可选参数,用来指定文件名的字符串表达式,可能包含目录或者文件夹以及驱动器,如果没有找到pathname, 则会返回零长度字符串(“”);
    • attributes可选参数.常熟或数值表达式,其综合用来指定文件属性.弱国省略会返回匹配pathname但不包含属性的文件
  • 删除
    • kill pathname 从磁盘中删除文件
    • RmDir pathname 从磁盘中删除目录
  • 打开 Open pathname For mode [Access access] [lock] As [#]filenumber [Len=reclength] 能够对文件输入/输出(I/O)
    • pathname必要字符串表达式,指定文件名,改文件名可能还包括目录,文件夹及驱动器
    • mode必要,和关键字,指定文件方式,有Append,Binary,Input,Output,或Random方式,如果未指定方式,Random访问方式打开文件
    • access 可选. 关键字,说明打开的文件可以进行的操作,有Read,Write或ReadWrite操作
    • Lock可选.关键字,说明现定于其他进程打开的文件的操作,有Shared,LockRead,LockWrite和Lock Read Write
    • filenumber 必要,一个有效的文件号,返回未1到511之间,使用Freefile函数可以得到下个可用的文件号,reclength可选. 小于或等于32, 767(字节)的一个数,对于用随机访问方式打开的文件,改值就是记录长度,对于顺序文件就是缓冲字符数
    • 在Binary，Input，Random方式下可以用不同的文件号打开同一文件，而不用先将该文件关闭。在append和Output方式下，如果要用不同的文件号打开同一文件，则必须在打开文件之前先关闭该文件。
  • 读入
    • Input #filenumber, varlist 从已经打开的顺序文件中读取数据并将数据指定给变量
    • Get [#]filenumber, [recnumber], varname 将一个已打开的磁盘文件读入到一个变量中
  • 写入
    • Write #filenumber, [outputlist] 将数据写入顺序文件
    • Print #filenumber, [outputlist] 将格式化显示的数据写入顺序文件中
    • Put [#]filenumber, [recnumber], varname 将一个变量的数据写入磁盘文件中
  • 关闭
    • Close [filenumberlist] 关闭Open语句打开的输入/输出（I/O）文件
  • 其他文件函数
    • LOF(filenumber) 返回一个Long，表示用Open语句打开的文件的大小，该大小以字节未单位
    • EOF(filenumber) 返回一个integer, 它包含Boolean值True, 表明已经到达未Random或顺序Input打开的文件的结尾
    • Loc(filenumber) 返回一个Long, 在一打开的文件中指定当前读/写位置
    • Seek(filenumber) 返回一个Long, 在Open语句打开的文件中指定当前的读/写位
