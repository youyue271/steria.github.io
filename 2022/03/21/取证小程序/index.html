取证小程序简介

  • 小程序使用python语言，通过公开的接口制作插件增加新的功能

  • python需要python3以上的版本

  • 取证小程序，在自动取证的地方有自定义取证的接口

  • 取证小程序目录结构

    (不仅取证大师,手机大师等都有)

    路径 作用
    FMP\Bin\Python37 32位的python运行环境,后续第三方库安装于此
    FMP\Bin64\Python37 64位的python运行环境,后续第三方库安装于此
    FMP\FmPlugins\Config\ 存放小程序管理的配置文件
    FMP\FmPlugins\Example\ 存放小程序的示例代码
    FMP\FmPlugins\Extends\ 存放功能扩展类小程序的代码
    FMP\FmPlugins\plugins\ 存放应用解析类小程序的代码
    FMP\FmPlugins\SDK\ 存放小程序的接口说明文档
    FMP\FmPlugins\Tools\ 存放取证小程序开发和运行过程相关的工具
    • 取证小程序的框架分为两大类:应用解析,和功能扩展
      • 应用解析框架提供一套基础的组件包含文件读取,数据入库和取证结果展示,能够创建新的应用解析小程序,并归入自动取证流程,为后续所有的硬盘实现一键分析
      • 功能扩展框架提供了一套基础组件,包括取证结果读取接口,界面控件,也支持导入第三方python库,实现更加复杂的小程序

编程基础—-常用函数和文件格式

常见文件格式

  • SQLite:相当于二维表
    • 导入: import _sqlite3 as sqlite3
    • 打开: connect(), cursor()
    • 执行: execute()
    • 获取: fetchall()
    • 关闭: close()
    • 常见于手机app的文件格式
  • XML文件格式(未定义标签文件):
    • 有对应的根元数,元数,属性(属性的值用双引号)
    • 用于传输和存储数据, HTML(有定义的标签文件)用于
    • 导入: import xml.etree.ElementTree as ET
    • 打开: ET.parse()
    • 根节点: getroot()
    • 查找:find()
    • 属性:tag, text
  • Json:轻量级数据交换
    • 常用于手机通信中
    • 导入: import json
    • 打开: json.loads()
    • 读取: 以数组方式读取, 如:student[‘id’]
  • 日志型文本:文本之间有一定规律
  • 复合文件:以二进制形式进行存储解析

常用函数

  • time函数
    • time和calendar模块可以用于格式化时间和日历(提取时间戳), 时间间隔是以秒为单位的浮点雄安舒
    • 导入: import time
    • 转换: time.localtime()
    • 转换: time.strftime()
  • base64
    • 导入:import base64
    • 编码:base64.b64encode
    • 解码:base64.b64decode
  • 进制
    • chr()返回整数i对应的ASCII字符
    • int()用于讲一个字符串或数字转换为整型
    • hex()转换一个整数对象为十六进制字符串表示
    • oct()转换一个整数对象为八进制的字符串表示
